欧雷流二三事

哈喽～既然来到了这里，相信你是想了解一些关于我或者本站的信息。不过遗憾的是，这里只有站点相关的，我的个人介绍在「欧雷」这个人物页中。

简要说明

本站的内容主要是生活、思想和技术类，并按照我的「多元性」将文章等信息做了聚合：

  • 思考者
  • 程序员
  • 御宅族

这些「聚合」在本站被称作「专题」，可以到专题列表查看全部。

推荐在进行一切操作之前先阅读《玩法手册》了解一下如何很好地使用本站。

站点变更

自成立以来时不时在更新，经历了几个阶段性的重大变革，无论是设计风格、访问速度，还是内容质量、阅读体验都得到了很大提升。

版本迭代

目前的架构和设计的版本为 Ourai.WS 4，主要使用 HTML5 和 CSS3 等技术进行开发，采用静态博客生成工具进行构建并部署到云主机上。在设计页面时，以提高阅读体验和保持简洁风格为最高设计原则，体现个性在其次。

本站的运行依赖于以下技术和服务：

  • 阿里云提供运行环境
  • DNSPOD 提供域名解析
  • Jekyll 生成静态站点
  • Disqus 提供评论功能
  • Share.js 提供社交分享

之前的网站架构所用过的技术：

  • Ourai.WS 3 采用 WordPress 进行开发

大事记

下面为在「欧雷流」设立后的主要事件：

  • 2009-05-01在博客托管服务平台博客大巴上建立个人博客「宅男的部屋」
  • 2012-02-23因受不了被服务提供商的条条框框所限制而将博客从博客大巴迁移到自己购买的虚拟主机上并注册了域名 ourai.ws
  • 2012-06-02设计 Ourai.WS 3.1 主题
  • 2013-02-15将与私生活相关的文章和相册重组成博客从主站分离出去
  • 2013-05-01设计 Ourai.WS 3.2 主题
  • 2013-05-15将分离出去的私人内容博客重新合并到主站中
  • 2013-05-28网站名称从「宅男的部屋」更改为「欧雷流」
  • 2015-05-07从虚拟主机迁移到 GitHub Pages
  • 2015-09-13启用 Ourai.WS 4 版本
  • 2016-03-03因百度无法收录 GitHub Pages 站点和考虑到网站的未来发展方向而决定迁移到阿里云
  • 2016-03-03开始对域名 ourai.ws 进行 ICP 备案
  • 2016-03-11域名 ourai.ws 通过 ICP 备案审核并将站点迁移到阿里云
  • 2016-03-21全站 HTTPS 访问
  • 2016-03-25支持按文章分类订阅 feed
  • 2016-06-29开始对域名 ourai.ws 进行公安备案
  • 2016-08-06域名 ourai.ws 通过公安备案审核
